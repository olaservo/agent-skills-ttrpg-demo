# agent-skills-ttrpg-demo

Demo of using **agent skills** and **MCP** for learning and playing tabletop RPGs — including
skill discovery and attribution prototypes.

- **`mcp/ttrpg-helper/`** — the MCP server (`ttrpg-helper-mcp`): a server-side dice resolver
  (animated UI), character-sheet UI resources, and a set of agent **skills** under
  `mcp/ttrpg-helper/skills/` for *Fallout: The Roleplaying Game* (Modiphius 2d20) and
  *Warrior, Rogue & Mage*.
