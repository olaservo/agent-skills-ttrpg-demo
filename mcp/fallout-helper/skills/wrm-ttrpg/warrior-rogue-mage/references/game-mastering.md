# Game Mastering

*This material is for the GM.*

## Running Warrior, Rogue & Mage

WR&M leaves a lot of room for interpretation, and in many cases **GM fiat plays a major role** -
an "old school" trait that makes the game a flexible toolbox. If something is missing that you
need, add it; if something bothers you, change it or drop it. The rules-light design lets you
make the game your own without fear of breaking it.

- **Start slow, then go epic.** Especially if you have not run WR&M before, start with modest
  challenges so players get a feel for the rules and the lethality of combat before you escalate
  to the epic fantasy (powerful artifacts, lost technologies, ruined cities) the setting
  promises.
- **GM rulings instead of rules.** When a rules question slows play, make a ruling and move on -
  the GM always has the last word. Look it up after the session if it still nags you.
- **Make it your own.** Add lands, create monsters, change rules, write your own spells. The book
  contains several optional rules, but house rules are encouraged too.
- **Set the mood with music.** Genre-appropriate soundtracks (film, game, or otherwise) greatly
  improve immersion.

## Character advancement

Characters in WR&M have **no levels and earn no experience points**. The **GM decides when
characters advance** - usually at the end of a successful adventure (or, if you prefer faster
growth, after each session or every other session).

When the GM allows an advance, the character chooses **one** of:

- **Raise one attribute by one** (recompute derived stats).
- **Add 1d6 to either HP or Mana.** *(Optional: add a flat 3 instead of rolling.)*
- **Gain an additional skill.**
- **Gain a talent.**

**Talents should be harder to earn** than a skill or HP. The GM should send the character on a
sidequest to find a trainer, gain admission to a special group, or learn an ancient ritual that
unlocks the talent.

## Non-combat hazards

Aside from combat, characters can be harmed in many ways. A few common hazards:

| Hazard | Damage |
|---|---|
| Fall | 1d6 per 3 yards fallen |
| Suffocation / drowning | 1d6 per round |
| Mild poison | 1d3 initial damage, then 1 per round until a successful Warrior check vs DL 7 |
| Lethal poison | 1d6 initial damage, then 2 per round until a successful Warrior check vs DL 11 |
| Fire | 1d6 per round exposed to the flames |

(Hazard damage, like all damage, follows the exploding-die rule where applicable. Remember a
character at under half max HP is *seriously wounded* if you use that optional rule - see
`combat.md`.)
